## Why this exists

Fusion Manage's API surface (v1, v2, and v3) is documented across scattered official docs, a Postman collection, and tribal knowledge from years of building PLM add-ins. This site reconciles those sources into one place — and every page is plain markdown, so an AI agent can read it directly from the repo without crawling rendered HTML. See `/llms.txt` and `/llms-full.txt` at the site root for a machine-readable index.
